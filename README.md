# AI Website Copy Generator
 Project Overview

This project demonstrates how AI can be used to generate website copy for local businesses using structured prompt engineering.

 Features

* Homepage content generation
* Service page descriptions
* Call-to-Action (CTA) writing
* Tone customization for different businesses

 Tools Used

* ChatGPT
* Google Docs / MS Word
* GitHub

 Prompt Engineering Approach

A structured prompt was designed to:

* Define business details
* Control tone and style
* Generate conversion-focused content

 Project Structure

AI-Website-Copy-Generator/
│
├── prompts/
├── outputs/
└── README.md

 Sample Output

Includes homepage, services page, and CTA content for a salon business.
